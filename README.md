# ReaperGL

ReaperGL is a small C++14 Engine project based on OpenGL 4.5 and later.

## Install

mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=$PWD/install ..
make install

Dependencies:
- [GLFW](github.com/glfw/glfw) 3.1+
- [glbinding](github.com/hpicgs/glbinding)
- [moGL](github.com/Ryp/moGL)
