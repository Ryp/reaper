# ReaperGL

ReaperGL is a small C++ Engine project based on OpenGL 4.4 and later.

## Install

mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=$PWD/install ..
make install

Dependencies:
- C++11
- GLFW 3+
- glbinding
