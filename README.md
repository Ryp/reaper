# ReaperGL

ReaperGL is a small C++ Engine project based on OpenGL 3.0 and later.

## Install

mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=$PWD/install ..
make install

Dependencies:
- C++11
- Glfw 3.0
- Glew 1.9.0
